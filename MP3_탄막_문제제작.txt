1. 플레이어 이동 ( 상 하 좌 우 이동)

	(1) 플레이어 생성	 - 플레이어의 비행선을 띄운다.
	(2) 키보드 입력		 - 유저가 키보드의 방향키를 누른다
	(3) 플레이어 이동	 - 캐릭터가 입력받은 방향으로 이동한다
	(4) 플레이어 이동	 - 플레이어의 위치가 개발자가 지정한 위치에서 나타난다
	(5) 플레이어 사망 	 - 플레이어의 비행선이 폭발한다(GAME OVER)
	(6) 플레이어 공격	 - 플레이어의 입력받은 키보드 방향으로 탄환(공격)을 발사한다
	(7) 플레이어 이동	 - 플레이어가 화면 밖으로 이동한다
	(8) 화면 이동 		 - 게임의 배경이 누르는 방향키에 따라서 이동한다.


2. 일직선 탄환 발사
	
	(1) 플레이어 생성 	- 플레이어의 비행선(혹은 적 비행선)을 띄운다
	(2) 키보드 입력 	- 유저가 키보드의 공격키(space)를 누른다
	(3) 탄환 생성 		- 비행선의 좌표에서 탄환을 생성한다
	(4) 탄환 이동		- 설정한 총알 속도, 방향으로 탄환이 이동한다
	(5) 탄환 제거		- 탄환이 생성된 후 제거된다
	(6) 플레이어 무적	- 플레이어가 총알을 발사할때 일시적으로 무적이 된다
	(7) 화면 이동		- 설정한 총알 속도, 방향으로 화면이 이동한다
	(8) 플레이어 이동 	- 플레이어의 비행선이 개발자가 지정한 위치에서 나타난다

3. 소용돌이 형태의 탄환 발사
	
	(1) 플레이어(적기) 생성	- 플레이어(적기)의 위치에 비행선을 띄운다		
	(2) 탄환 발사		- 탄환을 정해진 속도, 방향으로 이동시킨다		
	(3) 탄환 재생성		- 이전 탄환과 다른 각도값을 가지는 탄환을 발사한다	
	(4) 탄환 발사		- 다음 탄환을 발사한다
	(5) 탄환 회전		- 발사된 탄환을 360도 회전시킨다
	(6) 탄환 재생성		- 이전 탄환보다 빠른 속도를 가지는 탄환을 발사한다
	(7) 탄환 재생성		- 이전 탄환보다 생성되는 텀을 줄인다
	(8) 탄환 삭제		- 개발자가 정한 모양과 벗어나는 탄환은 제거한다

4. 여러 방향으로 발사되는 소용돌이 형태의 탄환 발사

	(1) 소용돌이 탄환 생성	- 앞서 공부한 소용돌이 형태의 탄막을 생성하는 탄환을 생성한다
	(2) 소용돌이 탄환 발사	- 앞서 공부한 소용돌이 형태의 탄막을 발사한다
	(3) 탄환 재생성		- 탄환의 각도값을 변화시킨 소용돌이 형태의 탄환을 생성한다
	(4) 탄환 재생성		- 탄환의 속도를 느리게 하여 소용돌이 형태의 탄환을 생성한다
	(5) 플레이어 이동	- 플레이어가 소용돌이 형태로 이동한다
	(6) 플레이어 사망	- 탄환이 화면 밖으로 나가면 플레이어가 사망한다
	(7) 화면 진동		- 탄환이 생성될 때 화면이 살짝 흔들린다
	(8) 탄막 시간 조절 	- 탄환이 발사된 이후 탄환의 시간을 조절하여 탄막의 간격을 조절한다

5. 여러 방향으로 동시에 탄 발사하기

	(1) 현재 탄 수 확인	- 적기의 탄 수가 2개 이상인지 확인한다
	(2) 탄환 생성		- 개발자가 임의로 지정한 탄막의 각도, 발사 범위의 넓이, 탄막의 수 값을 가지는 탄환을 생성한다
	(3) 탄환 발사		- 각 탄환이 가지는 속도만큼 이동하는 탄환을 발사한다.
	(4) 탄환 발사		- 탄환을 발사하는 기체를 중심으로 360도 모두 탄환을 발사한다
	(5) 적 기체 조준	- 적기의 위치 정보를 받아 그 방향을 향해 탄환을 생성하고 발사한다
	(6) 기체 생성		- 기체를 여러대 생성한다
	(7) 탄환 제거		- 발사된 탄환중 하나가 사라지면 같이 생성된 탄환을 모두 동시에 제거한다 
	(8) 탄환 발사		- 생성 되어있는 기체 모두 동시에 탄환을 생성하여 발사한다

6. 랜덤 방향으로 탄 발사하기

	(1) 탄환 생성		- 탄막의 각도나, 속도, 개수가 모두 랜덤으로 생성된다
	(2) 탄환 발사		- 각각 랜덤 값의 각도, 속도, 개수를 랜덤으로 가지는 탄환을 발사한다
	(3) 탄환 재생성		- 탄막의 각도나, 속도, 개수 모두 랜덤으로 재생성한다
	(4) 탄환 방향 수 변화	- 탄막을 재생성할 때 변수로 지정한 발사 방향의 수를 늘리거나 줄여서 탄환의 방향 수를 변화시킨다
	(5) 탄환 재생성		- 발사된 탄환이 새로운 탄환을 생성해낸다
	(6) 플레이어 위치 확인	- 플레이어의 탄을 발사할 때의 위치를 받는다
	(7) 화면 진동		- 탄이 발사될 때마다 화면이 흔들리도록 한다
	(8) 적기 무적		- 탄환을 발사할 때 해당 기체가 데미지를 입지 않도록 한다

7. 서로 다른 패턴을 가진 탄환을 동시에 발사하기

	(1) 패턴1 적기 생성	- 개발자가 지정한 패턴1의 탄막을 생성하는 적기를 생성한다
	(2) 패턴2 적기 생성	- 개발자가 지정한 패턴2의 탄막을 생성하는 적기를 생성한다
	(3) 패턴1과 2를 합치기	- 적기가 같은 위치에서 탄막을 생성한다
	(4) 이미지 처리		- 겹치는 이미지를 공백으로 처리한다
	(5) 패턴1과 1을 합치기	- 패턴1 적기 두개를 같은 위치에서 탄막을 생성하도록 한다
	(6) 패턴2 기체 사망	- 피격당해 기체가 파괴될 경우 패턴2를 사용하는 적기가 파괴된다
	(7) 탄막 제거		- 패턴2를 사용하는 즉시 패턴1의 탄막을 제거한다
	(8) 랜덤 탄 생성	- 랜덤 방향으로 탄을 발사한다

8. 유도탄
	
	(1) 최대 각도, 속도 설정	- 개발자가 지정한 속도나 각도보다 작은 속도로 꺾이도록 한다
	(2) 플레이어 위치 확인		- 탄을 발사하기 직전에 플레이어의 위치를 받는다
	(3) 탄막 생성			- 탄막이 해당 기체로부터 발사된다
	(4) 탄막 이동			- 탄막이 플레이어의 위치로 이동하지만 최대 각도, 속도보다는 느리게 꺾이도록 한다
	(5) 탄막 삭제			- 플레이어 있던 위치로 이동한 탄막은 없어진다
	(6) 탄막 이동			- 탄막이 플레이어의 위치로 이동한다
	(7) 플레이어 피격		- 탄막이 플레이어가 있었던 위치에 도착하면 플레이어의 체력이 감소한다
	(8) 최대 각도, 속도 설정	- 탄막이 이동하면서 최대 각도와 속도 설정을 변화 시킨다

	(5) 기체 분열		- 탄막과 충돌한 기체는 새로운 기체를 생성한다
